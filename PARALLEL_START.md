## ⚠️ **ВАЖНО:**

### **Возможна ситуация:**

Браузер откроется **ДО** того как backend полностью запустился.

Вы увидите:
```
GET https://localhost:5001/api/developers
ERR_CONNECTION_REFUSED
```

### **Решение:**

1. **Подождите 5-10 секунд**
2. **Нажмите F5** (обновить страницу)
3. ✅ Данные загрузятся!

ИЛИ просто подождите - Angular автоматически попробует снова через несколько секунд.

## 🔧 **ЕСЛИ ВИДИТЕ ERR_CONNECTION_REFUSED:**

### **Быстрое решение:**

1. Откройте окно "Backend API"
2. Дождитесь строки: `Now listening on: https://localhost:5001`
3. Вернитесь в браузер
4. Нажмите **F5**
5. ✅ Готово!

### **Если backend не запускается:**

```powershell
# Проверка
.\diagnose.bat

# Установка сертификата
dotnet dev-certs https --trust

# Перезапуск
Get-Process -Name "dotnet","node" -ErrorAction SilentlyContinue | Stop-Process -Force
.\start-app.bat
```
